# 🧠 Multi-Agent AI Research System (LangChain + LLMs)

🚀 A complete end-to-end **Multi-Agent AI Research Pipeline** built using **LangChain** and **Large Language Models (LLMs)** — implemented entirely in a single Jupyter Notebook.

This project demonstrates how to move beyond single prompts and build a system where **multiple AI agents collaborate** to perform real-world research tasks like searching, scraping, summarizing, and reviewing content automatically.

---

## 📌 Project Overview

This system simulates a **team of AI agents** working together:

* 🔎 **Research Agent** → Searches the web for relevant information
* 🌐 **Scraper Agent** → Extracts useful content from sources
* ✍️ **Writer Agent** → Generates structured reports
* 🧑‍⚖️ **Reviewer Agent** → Improves and validates the output

Together, they form a **fully automated research workflow**.

---

## 🎯 Features

* ⚡ Multi-agent orchestration using LangChain
* 🌍 Real-time web search integration
* 📄 Automated content extraction & summarization
* 🧾 AI-generated structured reports
* 🔁 Iterative review & refinement loop
* 🧠 Modular agent-based architecture
* 📓 Complete implementation in a single `.ipynb` notebook

---

## 🏗️ Architecture

```
User Query
     ↓
Research Agent (Web Search)
     ↓
Scraper Agent (Content Extraction)
     ↓
Writer Agent (Report Generation)
     ↓
Reviewer Agent (Validation & Improvement)
     ↓
Final Output
```

## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/atulkushwaha0112-py/Multi-Agent-Deep-Research-System.git
```

### 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

*(If not available, install manually: langchain, requests, etc.)*

### 3️⃣ Setup API Keys

Set your API keys (example):

```bash
export TAVILY_API_KEY=your_key_here
MISTRAL_API_KEY = api_key_here

```

### 4️⃣ Run the Notebook

```bash
jupyter notebook
```

Open:

```
main.ipynb
```
